# Modern Portfolio – Next.js + Tamagui (MVVM Architecture)

A sleek, vertically immersive portfolio built with **Next.js (App Router)** and **Tamagui**, inspired by a layered “hall climb” experience. Designed for performance, scalability, and clean architecture.

---

## Features

* Next.js App Router (fast, SEO-friendly)
* Tamagui UI System (theme + design tokens)
* Dark / Light mode support
* Fully responsive (mobile-first)
* Smooth vertical scrolling with layered animations
* MVVM Architecture (clean separation of concerns)
* Modular, reusable components
* Contact form with validation
---

## Animations

* Smooth scroll-based transitions
* Subtle parallax layering
* Lightweight motion (CSS / optional animation libs)

---

## Goals

* Clean, scalable architecture
* Easy content updates
* High performance
* Visually engaging UI
